# Escola Financeiro Completo

## Backend
Arquivos na raiz para deploy fácil no Render.

### Render
- Diretório raiz: vazio
- Comando de construção: npm install
- Comando start: node index.js

## Frontend
Pasta `frontend/` pronta para Vercel.

### Vercel
Variável:
`VITE_API_URL=https://SEU-BACKEND.onrender.com`
